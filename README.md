# RentNest Backend

A Rental House Management Backend API built with Express.js, TypeScript, Prisma ORM and PostgreSQL.

---

## Tech Stack

- Node.js
- Express.js
- TypeScript
- Prisma ORM
- PostgreSQL
- JWT Authentication
- bcrypt
- Swagger
- Postman

---

## Features

- Authentication
- Role Based Authorization
- Property Management
- Rental Request
- Payment Module
- Review Module
- Admin Dashboard
- Landlord Dashboard

---
